# Paper Skies 🛩️ — دليل النشر

موقع عالمي للفضفضة المجهولة: رسالتك تصير طيارة ورقية تدور حول الأرض 7 أيام ثم تختفي.

المعمارية (نفس نمط مشاريعك — صفر تكلفة):
- **الواجهة:** `index.html` على GitHub Pages
- **الباكند:** `Code.gs` على Google Apps Script + Google Sheet
- **الترجمة:** Anthropic API (المفتاح محفوظ في السيرفر فقط، لا يظهر أبداً في المتصفح)

---

## الخطوة 1 — الباكند (10 دقائق)

1. أنشئ Google Sheet جديد (سمّه مثلاً `paper-skies-db`)
2. من الشيت: **Extensions ← Apps Script**
3. احذف الكود الموجود والصق محتوى `Code.gs` كاملاً
4. من **⚙️ Project Settings ← Script Properties ← Add script property**:
   - Property: `ANTHROPIC_API_KEY`
   - Value: مفتاحك `sk-ant-...`
5. **Deploy ← New deployment ← Web app**:
   - Execute as: **Me**
   - Who has access: **Anyone**
6. انسخ رابط الـ Web app (ينتهي بـ `/exec`)

> عند أي تعديل مستقبلي على الكود: Deploy ← Manage deployments ← ✏️ ← New version

## الخطوة 2 — الواجهة (5 دقائق)

1. افتح `index.html` وابحث عن السطر:
   ```js
   const API_URL = '';
   ```
   والصق رابط `/exec` بين علامتي التنصيص
2. أنشئ مستودع GitHub جديد (مثلاً `paper-skies`) وارفع `index.html`
3. **Settings ← Pages ← Source: main branch** ← احفظ
4. موقعك: `https://abdullah5544.github.io/paper-skies/`

## ملاحظات

- **وضع تجريبي:** لو تركت `API_URL` فارغاً، الموقع يشتغل محلياً (رسائل تجريبية + حفظ في جهاز الزائر فقط) — مفيد للمعاينة قبل ربط الباكند.
- **التكلفة:** الترجمة الوحيدة التي تكلّف (هللات مع Haiku + الكاش). كل الباقي مجاني.
- **مراقبة المحتوى:** كل الرسائل في الشيت — احذف أي صف يدوياً ويختفي من الموقع خلال 90 ثانية (تحديث تلقائي).
